Questions

Before you install

Is ONFNS free?

Yes. The app is a free download on Google Play and you can browse the charts and profiles without creating an account.

Does it contain adult content?

No. ONFNS is a safe-for-work directory: no explicit media, no paywalled content and no private messaging. It publishes metadata and links to official public profiles.

How is the Hype Index calculated?

It blends web velocity, search trend data and fan voting momentum. Positions are recalculated continuously, so a strong voting day is visible on the chart.

How often can I vote?

Matchups refresh every 24 hours and each one takes a single vote per person. Come back tomorrow for a new pairing.

Is there an iPhone version?

ONFNS is on Android today. If an iOS build ships, it will be announced here first.

How do I request a profile or a correction?

Write to the contact address in the footer with the creator's official public links. Verified corrections and removal requests are handled first.
